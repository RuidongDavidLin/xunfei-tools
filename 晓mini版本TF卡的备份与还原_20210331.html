晓mini版本小车TF卡备份操作指南

1. TF卡取出，在PC上备份：

将晓mini的TF卡取出，使用TF卡读卡器接入PC电脑。

cd <clone_dir> # 进入PC中存放TF卡内容的目录
sudo -s
# 输入：PC的密码
cp -av /media/robot/_home/* ./ # 将TF卡中内容拷贝至存放目录 
# Ctrl + D 退出root

/media/robot/_home是TF卡在PC的挂载点，需根据df 或 lsblk指令出输出的挂载情况指定。

2. 在nano上直接向其他存储设备中备份

# 将U盘插入晓mini
lsblk # 查看U盘的挂载情况
cd <clone_dir_in_usb_disk> # 进入U盘中存放TF卡内容的目录
sudo -s
# 输入：ucar
cp -av /home/* ./ # 将TF卡中内容（即home中内容）拷贝至存放目录 
# Ctrl + D 退出root

TF卡还原：

1. 在PC上还原：

将待还原烧录的TF卡通过读卡器接入PC电脑。

  1. 进入root用户
sudo -s
  1. 解除TF卡挂载并格式化:

注意我这里的TF卡设备识别为/dev/sdc1，需要根据自己情况进行修改

umount -f /dev/sdc1
mkfs.ext4 /dev/sdc1 -L /home
  1. 创建PC上TF的临时目录并挂载TF卡:

注意我这里的TF卡设备识别为/dev/sdc1，需要根据自己情况进行修改

mkdir /mnt/tf_copy  
# /mnt/tf_copy也可以是自己指定的其他目录，但下文需要统一该目录名
mount /dev/sdc1 /mnt/tf_copy
  1. 将备份目录下的内容复制到临时挂载点中。
cd <clone_dir>
cp -av ./* /mnt/tf_copy
  1. 完成烧录，解除TF卡挂载并删除pc临时目录
umount -f /dev/sdc1
rm -rf /mnt/tf_copy # 可以不删除，看自己使用
